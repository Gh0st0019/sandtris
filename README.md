# Sandtris

Apri `index.html` in un browser moderno.

Controlli desktop:
- Frecce o WASD per muovere
- Z/X per ruotare
- Spazio per hard drop
- C per hold
- P o Esc per pausa

Controlli mobile:
- Swipe sinistra/destra per muovere
- Tap per ruotare
- Swipe in giu veloce per hard drop
- Pressione lunga per hold

Leaderboard:
- Salvata in localStorage su base settimanale.
